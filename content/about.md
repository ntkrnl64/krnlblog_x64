## 关于 KrnlBlog X64

这是一个基于 Fluent UI 的现代化静态博客系统。

### 技术栈

- **React** - 前端框架
- **Fluent UI** - Microsoft 设计系统
- **Vite** - 构建工具
- **TypeScript** - 类型安全
- **React Router** - 路由管理
- **Marked** - Markdown 解析
